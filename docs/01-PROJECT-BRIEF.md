# Braintrust Marketing Website — Project Brief

## Overview
Modern, AI-forward, enterprise-grade marketing website for Braintrust at `usebraintrust.com`. Complete rebuild serving three core audiences: Companies, Talent, and Token-curious users.

## Design Direction
- **Aesthetic**: Stripe-level polish, GPT/OpenAI clarity
- **Colors**: Braintrust brand colors (blue primary, teal accent, purple token)
- **Typography**: Inter font family (system fallback)
- **Expression**: Professional, technical, AI-forward. Not crypto-speculative.

## Three Audience Lanes

### 1. Companies (Clients)
- **Goal**: Hire talent, buy AI training data, use AI recruiting tools
- **Products**: Talent Marketplace, AIR (AI Recruiter), Nexus (Workflow Automation)
- **CTA**: Book a Demo → Salesforce lead routing

### 2. Talent (Job Seekers / AI Trainers)
- **Goal**: Find high-paying roles, earn money on AI training tasks
- **Value Prop**: Zero fees — keep 100% of earnings
- **CTA**: Sign Up → Talent portal

### 3. Token-Curious (BTRST)
- **Goal**: Understand BTRST utility, governance, tokenomics
- **Tone**: Utility-first, educational, compliant
- **CTA**: Learn more about governance/participation

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: CSS Modules + Global Design System
- **Hosting**: Vercel
- **SEO**: Static pre-rendering, meta tags, JSON-LD schema
- **API**: Next.js API routes for lead capture

## Key Metrics / Success Criteria
- Core Web Vitals: All green
- SEO: Unique meta per page, single H1, proper heading hierarchy, schema markup
- Lead Capture: Forms submit to API with Salesforce/Slack integration ready
- Accessibility: WCAG-aware, skip links, semantic HTML, ARIA labels
